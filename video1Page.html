What is a Sensor?

Defining Inputs

A sensor is a device that detects changes in the environment and sends the information to other electronics, frequently a computer processor.

Camera = Eyes

Visual Perception

Just like human eyes, cameras capture light to create images of the surrounding world.

Ultrasonic = Ears

Distance Measurement

Ultrasonic sensors work similarly to bats or dolphins, using sound waves to detect how far away an object is.

Step 1: Create or Login to Tinkercad

Go to Tinkercad and sign in or create a free account.

Step 2: Create a New Circuit

From the dashboard, click Circuits ‚Üí Create New Circuit.

Step 3: Add Components

Add the following components to your workspace:

  • Arduino Uno
  • Breadboard
  • Ultrasonic Sensor (HC-SR04)
  • Red LED
  • Resistor

Step 4: Connect Components

Wire the components as shown in the video. Ensure power (red) and ground (black) are connected correctly.

Step 5: Add Code

Click Code ‚Üí Text Mode and paste the Arduino code provided.

Step 6: Start Simulation

Click Start Simulation and observe the robot's behavior as you adjust the ultrasonic sensor.

Step 7: Adjust Distance and Observe LED

Move the distance slider on the ultrasonic sensor and observe the red LED:

  • Longer distance ‚Üí Red LED is OFF
  • Shorter distance ‚Üí Red LED is ON
