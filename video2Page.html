A* Algorithm Concept

Optimal path planning using cost + heuristic

A* (A-star) is a pathfinding algorithm that finds the shortest path from a start node to a goal node. It is widely used in robotics, AI, and navigation because it balances explored cost and estimated cost.

Cost Function

A* evaluates nodes using:

f(n) = g(n) + h(n)
  • g(n): actual cost from Start ‚Üí current node
  • h(n): heuristic estimate from current node ‚Üí Goal
  • f(n): total estimated cost via that node (A* picks the lowest f(n) next)
How A* works (high-level)
  1. Start from the Start cell and place it into an ‚Äúopen list‚Äù.
  2. Pick the node with the lowest f(n) from the open list.
  3. Expand its neighbours and update their costs if a better path is found.
  4. Move expanded nodes to a ‚Äúclosed list‚Äù and repeat until the Goal is reached.
Tip: If the heuristic h(n) never overestimates the remaining distance (admissible), A* is guaranteed to return an optimal shortest path.

Grid Map Representation

The environment is represented as a two-dimensional grid map, where each square corresponds to a possible position the robot can occupy.

  • Each grid cell represents a discrete location in the environment.
  • The robot can move between neighbouring cells in four directions: up, down, left, and right.
  • Some cells are blocked by obstacles and cannot be traversed.
